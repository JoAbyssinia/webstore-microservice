# webstore-microservice
This is a microservice based application for a Webstore.

Main Microservices in the project are:
1. customer-service
2. product-service
3. shoppingcart-command
4. shoppingcart-query
5. stock-service
6. order-service
7. Elk stack
8. kafka 
9. API gateway
10. kipkin
11. docker and docker-compose
